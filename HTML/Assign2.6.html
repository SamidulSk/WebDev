List of HTML Tags

  1. h1> to <h6> (Headings): Headings of different levels on a webpage.
  2. p> (Paragraph): Defines a paragraph of text.
  3. a> (Anchor): Creates hyperlinks within an HTML document.
  4. img> (Image): Inserts an image into a webpage.
  5. ul> and <li> (Unordered List and List Item): Creates an unordered (bulleted) list.
  6. ol> and <li> (Ordered List and List Item): Creates an ordered (numbered) list.
  7. div> (Division): A generic container element for grouping and organizing other elements.
  8. span> (Span): Inline-level element for applying styles to specific sections of text or content.
  9. form> (Form): Creates an HTML form for user input.
  10. input> (Input): Creates various types of input fields within a form.
