3 > Write a short note on tags, elements, and attributes along ith relevant examples>

 tags:- Tags in html used to define the structure and layout of web pages. 
        Tags are represented by angle brackets (<>) and usually come in pairs - an opening tag and a closing tag.
        examples:- <h1> tag to define the main heading of html document.

elements:- elements represented most basic component of html document,which can represent Text, List ,paragraph.
        example:-<p> element is used to define a paragraph of text. It has an opening tag <p> and a closing tag </p>.
         Any text or content placed between these tags will be displayed as a paragraph on the webpage.
        
attributes:- Attributes provide additional information about HTML elements.
             use for extending HTML element Behavior
             example:-<img src="example.jpg" alt="Example Image">
                "example.jpg" is the image source specified by the "src" attribute,
                and "Example Image" is the alternative text specified by the "alt" attribute.
